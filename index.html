Responsável: Ariele Couto — Neuropsicopedagoga (CBO: 2394-35)

2. Finalidade do Tratamento

Os dados coletados neste formulário têm como única finalidade a realização de triagem neuropsicopedagógica da criança indicada, para subsidiar o primeiro atendimento clínico presencial com a profissional Ariele Couto.

3. Dados que serão coletados

• Dados de identificação da criança (nome, data de nascimento, série escolar)

• Dados de contato do responsável (nome, e-mail, telefone)

• Dados sensíveis de saúde e desenvolvimento (histórico de desenvolvimento, comportamento, aprendizagem, aspectos emocionais, sono, diagnósticos anteriores e medicações)

Por envolver dados de menor de 18 anos, este tratamento está sujeito ao Art. 14 da LGPD, que exige proteção reforçada.

4. Base Legal

Tratamento realizado com base no consentimento do titular (Art. 7º, I da LGPD) e na tutela da saúde do menor (Art. 11, II, f da LGPD).

5. Compartilhamento de Dados

Os dados são transmitidos de forma segura via Formspree (processador de formulários GDPR-compliant) exclusivamente ao e-mail profissional da Neuropsicopedagoga. Não há compartilhamento com terceiros para fins comerciais.

6. Retenção e Eliminação

Os dados serão mantidos pelo prazo de 5 (cinco) anos após o encerramento do acompanhamento, conforme recomendação do Conselho Federal de Psicopedagogia, e então eliminados de forma segura.

7. Seus Direitos (Art. 18 LGPD)

  • Confirmação de existência de tratamento
  • Acesso aos dados
  • Correção de dados incompletos ou inexatos
  • Anonimização, bloqueio ou eliminação de dados desnecessários
  • Portabilidade dos dados
  • Eliminação dos dados tratados com consentimento
  • Revogação do consentimento a qualquer momento

Para exercer qualquer direito, entre em contato: arielecouto@email.com.br

8. Segurança

Os dados são transmitidos via HTTPS com criptografia TLS. O formulário não armazena dados em banco de dados próprio — os dados chegam diretamente ao e-mail profissional da responsável pelo tratamento.
